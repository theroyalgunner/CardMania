# CardMania V7.2 Portfolio Pro

This version builds on V7.1 and keeps the cloud-first Supabase flow.

## Setup
1. Copy your existing `.env.local` into this folder.
2. Run:

```bash
npm install --registry=https://registry.npmjs.org/
npm run dev
```

## New in V7.2
- Portfolio Pro dashboard
- Value by manufacturer
- Value by team
- Top profit cards
- Cards needing cleanup
- Better ROI / P&L insights

## Important
`.env.local`, `.next`, and `node_modules` are not included for safety and file size.
